Site Residencial Clone

Desenvolvi um site residencial clone utilizando HTML, CSS, Bootstrap e ReactJS.
Este projeto foi um excelente exercício para aprimorar minhas habilidades em front-end e aprofundar meus conhecimentos em ReactJS.

 Tecnologias Utilizadas

HTML5: Estruturação do conteúdo da página

CSS3 & Bootstrap: Criação de um design atrativo, responsivo e consistente

ReactJS: Desenvolvimento de uma interface interativa e dinâmica

 Hospedagem

O site está hospedado na Netlify, garantindo carregamento rápido e uma experiência de usuário fluida.

 Confira o site aqui: Acesse o site

 Objetivos do Projeto

Praticar HTML, CSS e Bootstrap para criar layouts responsivos

Aprender a integrar ReactJS em projetos front-end

Criar um clone funcional e esteticamente agradável de um site residencial

 Observações

 Projeto focado apenas no front-end

Ideal para estudo e portfólio

Permite aprimorar conhecimentos em componentes React e design responsivo
